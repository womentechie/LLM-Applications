# !! INTEL MAC (x86_64) WITH PYCHARM VENV — READ FIRST !!
# ---------------------------------------------------------
# On Intel Mac, `pip` may point to the system Python (/usr/local/bin/pip)
# even when your venv is activated. Always use pip3 or python -m pip
# to ensure packages install into your project venv, not system Python.
#
# Verify before installing:
#   which pip3    → should show .venv/bin/pip3   ✅
#   which python  → should show .venv/bin/python ✅
#   which pip     → may show /usr/local/bin/pip  ⚠️  (use pip3 instead)
#
# INSTALL ORDER — run these commands exactly in this sequence:
#
#   Step 1:  pip3 install "onnxruntime==1.19.2"
#   Step 2:  pip3 install chromadb
#   Step 3:  pip3 install langchain-chroma
#   Step 4:  pip3 install -r requirements.txt
#   Step 5:  pip3 install faiss-cpu --no-deps
#   Step 6:  python -c "import chromadb; import faiss; print('✅ Both working')"
#
# WHY THE ORDER MATTERS:
#   onnxruntime >= 1.20 has no Intel Mac (x86_64) wheel — pin 1.19.2 first
#   so pip3 never tries to upgrade it when resolving chromadb's dependencies.
#   faiss-cpu pulls in numba/llvmlite which fail to build on Intel Mac —
#   --no-deps skips those, and faiss still works fine with just numpy.
#
# Apple Silicon (arm64) users:
#   pip3 install -r requirements.txt
#   pip3 install faiss-cpu --no-deps
#   python -c "import chromadb; import faiss; print('✅ Both working')"
# ============================================================
